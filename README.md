## Getting Started

Clone repository:
```bash
git clone git@github.com:PMGH/react-component-library.git
```

Node version:
```
12
```

Install dependencies:
```bash
yarn install
```

Run styleguide:
```bash
npm run styleguide
```

Navigate to localhost:6060
